# Riki — 리스크 및 가정 감사

## 리스크 목록

### R-01: 세션 시작 시 role memory 미로드 🔴
- **가정**: "세션 체크리스트를 따르면 역할이 컨텍스트를 유지한다"
- **실제**: 체크리스트에 role memory 로드 항목이 없음 → 구조적으로 불가능
- **결과**: 매 세션 역할이 빈 상태에서 시작. 학습이 저장되어도 다음 세션에 전달 안 됨 (E-012)
- **수복**: CLAUDE.md 세션 시작 step 4 추가 → session_009에서 수복됨

### R-02: 미발언 역할의 memory 갱신 모호성 🟡
- **가정**: "발언한 역할만 memory를 갱신하면 된다"
- **문제**: '발언'의 기준이 불명확 → 체크리스트 항목 6이 실질적으로 skip됨
- **현재 상태**: CLAUDE.md에 "Roles that did not speak may be skipped (not a gap)" 추가로 명문화

### R-03: 소급 수정이 재발 방지를 대체할 위험 🟡
- **패턴**: 이번 세션은 소급(backfill)으로 해결. 소급 자체는 좋으나, 구조 변경 없이 소급만 하면 다음 세션에서 동일 문제 반복
- **감시**: 다음 세션 시작 시 role memory 로드 여부로 재발 여부 확인 가능
- **tripwire**: "세션 시작 시 role memory를 읽지 않으면 → 학습 순환 고리 단절 재발"
