# Nova — COPD Track B 다른 관점 제시 (speculative)

Master 명시 호출(session_034). Ace가 G2 실패 → Pooled 강등으로 수렴한 결론에 대해, 질문 자체를 재설정하는 4가지 궤도를 제시.

## 궤도 1 — PRISm을 Outcome으로 뒤집기 🔄

- **Nova 프레임**: edu → PRISm 상태 자체 (진단 여정의 병목)
- **근거**: PRISm은 pre-COPD 상태. SES가 "COPD 진행 vs 정상 회복" 중간 머무름을 만들 가능성.
- **데이터 가능성**: KOCOSS FU_Y1~Y5 spirometry 있으면 transition analysis 가능 (COPD 진행/정상 회복/PRISm 유지 3-way)
- **타겟 저널**: *Eur Respir J*, *Chest* — trajectory 서사 선호
- **Salami 방어 강화**: Track A(COPD 확진)와 outcome이 명확히 다름
- **Ace 판정**: 🟡 **유보 (별도 Paper B 씨앗)**. 이번 토픽 scope 초과. PD-012로 등록.

## 궤도 2 — "매개분석"을 "수정가능 중재 타겟 발굴"로 재프레임 🎯

- **Nova 프레임**: 통계는 동일, 서사를 "설명"에서 "처방"으로 전환
- **핵심**: SGRQ Activity subscale이 rehabilitation으로 modifiable. Pack-years는 과거 누적이라 modifiable 아님.
- **타겟 저널**: *Lancet Respir Med* (IF 38) — translational 서사
- **비용**: Discussion·Abstract 문장 전환만 (+3~5일)
- **Ace 판정**: 🟢 **채용** (수위 조절: "intervention prioritization" → "modifiable mediator identification"). Phase 7 필수 작업.

## 궤도 3 — 한국 보편의료를 "Natural Laboratory"로 재포지셔닝 🇰🇷

- **Nova 프레임**: KO1_F_AIDS 거의 상수(제약)를 "의료접근 격차가 통제된 자연실험실(강점)"로 역전환
- **근거**: 미국·영국 연구는 SES→의료접근→질병 경로 지배적. 한국 보편의료에서 이 경로 자연차단 → 건강행동·문해력·증상보고 순수 효과 식별 가능
- **타겟 저널**: *Int J Epidemiol*, *Am J Epidemiol*, *Social Science & Medicine* (사회역학) IF 9~13
- **비용**: Intro 1문단 + Cover letter 1문장 (+2~3일)
- **Nova 확률 평가**: +20%p (가장 높은 ROI)
- **Ace 판정**: 🟢 **채용** (수위 조절: "natural experiment" → "natural laboratory" / "context with minimized access barriers"). Introduction·Cover letter·Limitation에 반영.
